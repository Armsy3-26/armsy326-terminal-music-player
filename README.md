2.open terminal.\
3.run: *gedit ~/.bashrc*\

4.add: *alias armsy326="/upper_path_leading_to_alixe_directory/Alixe/main.py"*\

5.save the .bashrc file and exit.\
6.run *armsy326* in the terminal.\
